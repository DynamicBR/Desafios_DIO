A Históra Da Web

Primeiros Computadores

Primeira Geração

O primeiro computador criado foi o MARK 1. Surgiu no ano de 1944. Seu principal intuito foi realizar grandes operações matemáticas em menos tempo.

Colossus e ENIAC sugiram após o MARK 1. No de 1946, no período da segunda guerrda munidal. Eles tinham como objetivo realizar criptoanálises e calcular a posição do inimigo. Colossus cooperava ao lado da Alemanha enquanto ENIAC cooperava aos Estados Unidos.

Segunda Geração

A segunda geração durou por volta de 1959 à 1965. Seu principal quesito foi a subistuição de válvulas para os transitores. Consequentemente diminuido drásticamente o tamanho do computador. Agora em vez de ocupar uma sala inteira como os da primeira geração. Você consegue coloca-lo em uma mesa e mudar de uma sala para outra.

Terceira Geração

Esta durou por volta dos períodos 1965 até 1970. Ela revolucionou com a chegada dos circuitos integrados, diminuindo ainda mais os tamanhos e popularizando os computadores como conhecemos hoje.

Quarta Geração

Esta é a geração atual de computadores. Esse período revolucionou com as chegadas dos periféricos customizados e grande quantidade de computadores criados para uso pessoal e a chegada dos jogos digitais. Aumentando drásticamente o seu desempenho.

Como surgiu a Internet

A primeira comunicação foi o telégrafo. Criado em 1958 e o uso do código morse para operações mais específicas e discretas.

Na década de 1950 na guerra fria os soldados precisavam se comunicar. Então a instituição DARPA elaborou os primeiros projetos da internet junto do ENIAC.

Joseph Carl Robnett Licklider teorizou sobre uma rede onde todos poderiam se comunicar através dela. A partir deste pensamento, diversos grupos de pesquisadores começaram a investir nessa ideia. Daí que surgiu a internet e o serviço nuvem.

Arpanet foi a instituição que começou a criar sistemas de comunicação em pacotes enviados um à um. Permitindo múltiplos destinos. Isso seria um pre esboço dos primeiros roteadores. O primeiro teste realizado teve participação de duas faculdades dos estados unidos.

O que são Clients?

Conceito de Client no Linguajar Web

Cliente é o consumidor de alguma informação na internet. Ele se conecta a Internet(protocolo http) que está hospedado em um servidor.

Quando nós estamos conectados a internet ela nos retorna as informações: HTML, CSS e JS/TS e arquivos estáticos.

Armazenando estes arquivos temporiamente no Cache. Para que, quando o usuário entre novamente no mesmo site, ele não tenha que baixar novamente as informações.

Quando se estamos trabalhando com conceitos de client. Precisamos estar atento as resoluções. Pois à diversos tipos de dispositivos de telas na hora da visualização de website. Chamamos isto de Design Responsivo.

Também prestar atenção a compatibilidade entre os navegadores com boas práticas e testes. Pois nem todos os comportamentos dos navegadores são padronizados.

Navegadores

Aplicações web são soluções criadas que possuem a internet como meio de comunicação entre Client x Server. Não sendo necessário à sua instalação.

Uma aplicação web é todo aquele que é acessado através do navegador. Caso você precise instalar um software para você usar em seu dispositivo pessoal, ele deixa de ser uma aplicação web.

O que são Servers?

Servidores é o lugar onde é hospedado a aplicação web junto do banco de dados. Para ele se comunicar com o client é usado o LAN(Local Area Network).

Hardware e Softwares em Servidores

Um servidor pode ser separado em duas categorias

Lógica que seria a parte do Software que englobam estes itens: Sitemas Operacionais Monitoramento Serviodores Web

Física que é a parte do Hardware. Ele está relacionado à: Armazenamento Memória Processadores

Tipos de Servidores

Proxy

É um servidor interno para proteção de sites maliciosos. Muito utilizado para empresas que protegem seus funcionários de malwares ou golpes comuns de internet.

FireWall

Um servidor externo de segurança utilizado também para proteção de sites maliciosos.

Web Server

Servidors online para armazenar e transmitir dados em tempo real. Como: Internet email Database Application Server Você também pode usar um servidor web sem internet. Chamamos de Intranet. Ela só pode ser usada com LAN.

DNS - Domain Name Service

Um servidor que redireciona com IPs para cada client acessando seu serviço. Em outras palavras, cada computador terá o seu próprio DNS. Para conectar no servidor ou outros se conectaram a ele.

Hospedagem de Sites

Hospedagem é um servidor para você alocar sua aplicação web. Para vocÊ mandar arquivos da sua aplicação à sua abordagem, usa-se a conexão FTP(File Transfer Protocol)

Linguagens de Progamação

O que é uma linguagem de progamação?

É uma linguagem escrita e formal que específica um conjunto de instruções e regras usadas para gerar softwares. Exemplos: PHP TypeScript/JavaScript

Linguagem Server-Side

Linguagens server-side, são linguagens que executam no servidor. Exemplos: PHP Java C# Ruby NodeJS

Linguagens Client-Side

Linguagens que são executadas apenas no cliente. Exemplos: HTML CSS JavaScript/TypeScript
